Welcome to the Movie API

This API allows users to access a collection of movies, including details like title, year of release, genre, and director. Use the endpoints below to interact with the data.

Endpoints

  • GET /movies
    Description: Returns a JSON array with a list of movies.
    • Example Response:
      •           [
            {
              "title": "Inception",
              "description": "A mind-bending thriller where a thief enters dreams.",
              "genre": {
                "name": "Sci-Fi",
                "description": "Science Fiction"
              },
              "director": {
                "name": "Christopher Nolan",
                "bio": "British-American filmmaker known for thought-provoking films with complex narratives.",
                "birthYear": 1970,
                "deathYear": null
              },
              "imageURL": "https://example.com/inception.jpg",
              "featured": true
            },
            {
              "title": "The Matrix",
              "description": "A hacker discovers the truth about reality in a simulated world.",
              "genre": {
                "name": "Sci-Fi",
                "description": "Science Fiction"
              },
              "director": {
                "name": "Lana Wachowski",
                "bio": "American filmmaker and writer.",
                "birthYear": 1965,
                "deathYear": null
              },
              "imageURL": "https://example.com/matrix.jpg",
              "featured": false
            }
          ]
  • GET /movies/:title
    Description: Returns data about a specific movie by title.
    • Parameters: title (string)
    • Example URL: /movies/Inception
    • Example Response:
      •           {
            "title": "Inception",
            "description": "A mind-bending thriller where a thief enters dreams.",
            "genre": {
              "name": "Sci-Fi",
              "description": "Science Fiction"
            },
            "director": {
              "name": "Christopher Nolan",
              "bio": "British-American filmmaker known for thought-provoking films with complex narratives.",
              "birthYear": 1970,
              "deathYear": null
            },
            "imageURL": "https://example.com/inception.jpg",
            "featured": true
          }
  • GET /movies/genre/:genre

  • POST /users
    Description: Allows a new user to register.
    • Expected Body:
      •           {
            "username": "newuser",
            "password": "userpassword",
            "email": "user@example.com",
            "birthday": "1990-01-01"
          }
    • Example Response: Confirmation message
